Quick Fix for Stuck Sidebar

If your sidebar is stuck collapsed and won't expand, use the buttons below to reset it:

Alternative Methods:

  • Keyboard Shortcut: Press Ctrl + B to toggle
  • Double-click: Double-click on the sidebar to toggle
  • Debug Page: Visit debug_sidebar.php
  • Browser Console: Press F12 and run the commands below

Manual Console Commands:

// Expand sidebar localStorage.setItem('sidebarCollapsed', 'false'); location.reload();
// Collapse sidebar localStorage.setItem('sidebarCollapsed', 'true'); location.reload();
// Clear all sidebar data localStorage.removeItem('sidebarCollapsed'); location.reload();
